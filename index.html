Mentions Légales

1. Présentation du site

Le site MyAim est une réalisation technique personnelle à vocation de démonstration. Il s'agit d'un prototype fonctionnel illustrant des compétences en développement web et en conception d'outils pour le BTP.

Développeur & Éditeur : Ivan HOARAU
Statut : Étudiant en BTS Maintenance des Systèmes Énergétiques et Fluidiques (SFE).
Contact : contact.hoarauivan@gmail.com

2. Hébergement

Ce site est hébergé à titre gratuit sur une solution de déploiement statique (GitHub Pages / Vercel / Local) dans le cadre d'un portfolio étudiant.

3. Propriété Intellectuelle

La structure générale, les textes, images animées et les outils de calcul (logique SFE) sont la propriété exclusive d'Ivan Hoarau. Toute reproduction totale ou partielle de ce site sans autorisation expresse est interdite.

4. Données Personnelles

Ce site est une démonstration technique (Front-End). Aucune base de données serveur n'est connectée. Les informations saisies sont stockées uniquement localement sur votre appareil (LocalStorage) pour simuler le fonctionnement de l'application.

Confidentialité
